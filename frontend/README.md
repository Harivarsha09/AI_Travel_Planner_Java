# AI Travel Planner

An AI-powered travel planning application built using React.js and Spring Boot.

## Tech Stack

### Frontend
- React.js
- Vite
- Bootstrap
- Axios

### Backend
- Java
- Spring Boot
- MySQL

## Features

- User Registration
- User Login
- AI Trip Planner
- Destination Recommendations
- Budget Planning
- Travel Dashboard

## Project Status

Frontend UI completed.
Backend development in progress.
